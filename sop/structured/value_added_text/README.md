# Value-Added Service Text Blocks

This directory is generated by `scripts/build_value_added_text_blocks.py`.

## Scope

- Includes:
  - `sop/债务咨询顾问服务费（还款无忧）场景（12.11更新）/`
  - single-service `.xlsx` files under `sop/增值服务大全（未上翻）/`
- Skips by default:
  - `sop/增值服务大全（未上翻）/增值业务大全20260105 (2).xlsx`

The skipped workbook is treated as a duplicate summary workbook. Add it later only as a fallback source if a service has no single-service file.

## Files

- `services.json`: service/product alias index.
- `text_blocks.jsonl`: weakly structured text blocks with source provenance.
- `manifest.json`: generation summary and diagnostics.

## Text Block Shape

Each line in `text_blocks.jsonl` is one retrievable evidence block:

- `service_id`, `service_name`, `service_aliases`
- `source_file`, `sheet`, `row_range`
- `block_type`: `qa_row`, `sheet_rows`, or `docx_paragraphs`
- `intent_tags`: coarse tags such as `inquiry`, `cancel`, `refund`, `bill_repayment`, `operation_path`
- `text`: source-preserving text
- `embedding_text`: text for retrieval
- `display_text`: text for prompt/reference display

This stage does not process embedded images. Image OCR/caption blocks should be added as a separate stage.
