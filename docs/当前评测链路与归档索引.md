# 当前评测链路与归档索引

更新时间：2026-05-19

这份文档是当前评测链路的入口索引。历史材料已经归档到 `archive/20260519_eval_chain_cleanup/`，日常测试优先看本文件和 `tests/EVAL_RUNBOOK.md`。

## 当前目录结构

```text
fin_copilot/                       # 主链路代码
skills/                            # Skill registry、YAML 定义、Router/生成 prompt
sop/                               # SOP 与结构化知识来源
rules/                             # Chain A 规则短路配置
tools/                             # Mock 业务工具
scripts/                           # 数据构建、评测、审计辅助脚本
tests/eval/                        # 离线评测脚本
tests/unit/                        # 单元测试
tests/reports/                     # 只保留当前可用报告
archive/20260519_eval_chain_cleanup/ # 历史文档、报告、中间产物归档
```

## 当前核心文件

| 类型 | 路径 | 说明 |
|---|---|---|
| 多轮 golden | `golden_test.jsonl` | 295 通电话，call 级 `gold_intents + queries`；只补了原始空 gold 的 2 通 |
| 旧单 query 测试集 | `raw_test.jsonl` | 2846 条单 query，字段为 `query/gold_skill/confidence` |
| 原始电话数据 | `原始300条数据.jsonl` | 多轮电话原始来源，原 `merged.jsonl` 改名后文件 |
| 旧单 query 数据快照 | `3000条raw data.jsonl` | 当前与 `raw_test.jsonl` 内容一致 |
| 标注维度 | `标注维度.xlsx` | golden 小结类别的来源维度 |
| 小结映射表 | `scripts/references/merged_intent_skill_mapping.json` | `gold intent -> 一个或多个 skill_id` |
| 映射审计文档 | `docs/小结标注_sop映射错配审计.md` | 小结类别与 SOP/skill 的对应关系审计 |
| 特殊标注样本 | `docs/golden小结类别_sop特殊标注全量样本.json` | 特殊类别的全量样本证据 |
| 最新逐条复核 | `docs/golden小结类别_sop逐条复核_20260519.md` | 基于当前 golden 和旧预测结果的逐条计分 |
| 高优先级疑点 | `docs/golden小结类别_sop高优先级疑点_20260519.md` | 只作为人工确认清单，不写回 golden |

## 当前评测入口

### 多轮真实电话评测

```bash
python3 tests/eval/merged_multi_turn_skill_recall.py --route-mode router --concurrency 8
```

输入：

- `golden_test.jsonl`
- `scripts/references/merged_intent_skill_mapping.json`

输出：

- `tests/reports/merged_multi_turn_<timestamp>/query_predictions.jsonl`
- `tests/reports/merged_multi_turn_<timestamp>/call_scores.jsonl`
- `tests/reports/merged_multi_turn_<timestamp>/summary.json`

当前复用的历史预测结果保留在：

- `tests/reports/merged_multi_turn_after_corporate_repay_tuning_20260427/query_predictions.jsonl`

当前映射重算结果保留在：

- `tests/reports/merged_mapping_recalc_from_previous_20260519/summary_after_empty_only_patch.json`

### 旧单 query golden 评测

```bash
bash scripts/run_golden_full_eval.sh
```

输入：

- `raw_test.jsonl`

主要脚本：

- `tests/eval/exp1_l1_domain.py`
- `tests/eval/exp2_skill_match.py`
- `tests/eval/exp3_chain_distribution.py`

### 分支/工具流评测

```bash
python3 scripts/eval_real_multiturn_branch_selection.py
python3 scripts/eval_real_query_branch_selection.py
python3 scripts/eval_tool_flow_branch_selection.py
```

这些脚本用于诊断 branch 选择或工具流，不是主准确率口径。
